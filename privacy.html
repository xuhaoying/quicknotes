QuickNote Web Clipper Privacy Policy

Last updated: March 2024

Data Collection

QuickNote Web Clipper does not collect or transmit any personal data. All notes and data are stored locally on your device using Chrome's storage API.

Permissions Used

The extension requires the following permissions to function:

  • storage: Used to save your notes locally on your device.
  • activeTab: Used to access the current tab when you select text to create a note.
  • scripting: Used to dynamically inject the note-taking interface and styles into web pages. This permission is essential for the following reasons:
    • To create the floating note input interface when you select text
    • To apply custom styles to the note interface without affecting the webpage's original styling
    • To ensure the note interface appears correctly on all websites
    • To maintain a consistent user experience across different web pages
    The extension only uses this permission when you actively interact with it (clicking the extension icon or selecting text), and it never runs scripts in the background or without your explicit action.

Data Storage

All notes are stored locally on your device using Chrome's storage API. The extension does not have access to any external servers or cloud storage.

Data Sharing

The extension does not share any data with third parties. All data remains on your device and is only accessible to you.
